# 🌍 Global Health Spending Dashboard

Análise exploratória e visualização dos gastos globais com saúde baseada nos dados do **GHED (WHO Global Health Expenditure Database)**.

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/meu-projeto-saude.git](https://github.com/seu-usuario/meu-projeto-saude.git)
   cd meu-projeto-saude
   ```
2. **Crie e ative um ambiente virtual:**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
3. **Instale as dependências**
   ```bash
   pip install -r requirements.txt
   ```
4. **Execute o Dashboard Streamlit**
   ```bash
   streamlit run app.py
   ```
